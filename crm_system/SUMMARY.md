## 🎯 核心功能模块

### 模块1: 客户增长漏斗系统
- **ToFu（引流层）**: 接粉、建群、留群、初筛
- **MoFu（培育层）**: 意向评分、粘度指数、开户管理
- **BoFu（转化层）**: 入金管理、出金预警、LTV预测

### 模块2: CRM客户管理
- 360度客户画像
- 智能标签系统（30项意向信号）
- 客户旅程可视化（9阶段）
- 聊天记录管理
- AI下一步动作推荐

### 模块3: 话术与素材库
- 12大模型话术库
- 四大黄金链路（反问/粘度/转化/首金）
- 场景智能匹配
- 素材发布系统

### 模块4: 员工运营系统
- 每日任务自动生成
- 打卡与行为追踪
- 个人KPI看板
- 反问次数统计

### 模块5: 培训与考核
- 7天/30天/90天培训路径
- 在线课程系统
- AI对话模拟
- 考试认证

### 模块6: SOP执行引擎
- 智能推荐下一步
- 话术自动匹配
- 流程合规检查

### 模块7: 数据仪表板
- 管理者驾驶舱
- 三层漏斗可视化
- 员工KPI雷达图
- 趋势预测

### 模块8: 风控预警
- 出金风险评分
- 情绪词监控
- 异常行为检测

### 模块9: 团队协作
- RBAC权限管理
- 客户分配
- 消息通知

## 🛠️ 技术架构

### 前端技术栈
- **框架**: React 18 + Next.js 14
- **UI**: Ant Design 5
- **状态**: Zustand
- **图表**: ECharts 5
- **移动端**: React Native / UniApp

### 后端技术栈
- **运行时**: Node.js 20 LTS
- **框架**: NestJS
- **语言**: TypeScript 5
- **ORM**: Prisma
- **认证**: JWT + Passport.js

### 数据库
- **主库**: PostgreSQL 14+
- **缓存**: Redis 7+
- **搜索**: ElasticSearch 8+
- **分析**: ClickHouse
- **存储**: MinIO / AWS S3

### 中间件
- **消息队列**: RabbitMQ
- **API网关**: Kong / Nginx
- **服务发现**: Consul
